## 📖 Overview · 概览

This repository documents my continuous learning journey in **Java backend development**. It covers core fundamentals, mainstream frameworks, distributed systems, DevOps practices, and real-world project examples. The content is organized to help both **job seekers** (domestic/international) and **graduate school applicants** demonstrate systematic thinking and engineering depth.

本仓库记录了我的 Java 后端持续学习历程，涵盖核心基础、主流框架、分布式系统、DevOps 实践以及真实项目示例。内容组织旨在帮助 **求职者**（国内外）和 **升学申请者** 展示系统性思维与工程深度。

## 🗂️ Repository Structure · 目录结构

java-backend-roadmap/

├── java/                    # Core Java (集合、JVM、并发、IO/NIO)

├── spring/                  # Spring Ecosystem (Boot, MVC, AOP, Transactions)

├── database/                # MySQL (SQL优化、索引、锁机制)

├── cache/                   # Redis (缓存策略、分布式锁、持久化)

├── middleware/              # Middleware (RabbitMQ, Kafka)

├── distributed/             # Distributed Systems (ID生成、分布式事务、一致性)

├── microservices/           # Microservices (Spring Cloud: Nacos, Gateway, Sentinel)

├── linux/                   # Linux Commands & Shell Scripts

├── docker/                  # Docker Basics & Compose

├── kubernetes/              # Kubernetes Concepts & Deployment

├── algorithm/               # Data Structures & LeetCode Solutions

├── design-patterns/         # 23 Design Patterns with Spring Examples

├── network/                 # Network Basics (TCP/IP, HTTP, Netty)

├── system-design/           # System Design Cases (秒杀、短链、IM)

├── monitoring/              # Monitoring (ELK, SkyWalking, Prometheus)

├── test/                    # Testing (Unit, Integration, Performance with JMeter)

├── projects/                # Full-Stack Demo Projects

└── tools/                   # Development Tools (Git, Maven, CI/CD)

Each folder contains its own `README.md` with key points, code snippets, and runnable demos.  
每个子目录都有独立的 README，包含要点、代码片段和可运行的示例。

## ✨ Why This Repository? · 为什么要建这个仓库？

### For Job Interviews · 面试准备
- Covers frequently asked topics in **senior Java engineer** interviews (concurrency, JVM tuning, distributed transactions, etc.).
- Provides structured notes and hands-on demos to reinforce understanding.

### For Overseas Study · 出国留学
- Demonstrates **self-driven learning** and **English documentation ability** – highly valued by foreign universities and companies.
- Shows familiarity with modern tech stacks (Cloud Native, Microservices).

### For Personal Growth · 个人成长
- Forces deep learning through writing and coding.
- Builds a portable portfolio that evolves with your career.

## 🚦 How to Use · 如何使用

1. **Browse** the directory that matches your current learning focus.
2. **Read** the `README.md` inside each folder for summary and key takeaways.
3. **Run** the code demos locally (JDK 17+, Maven 3.8+, Docker optional).
4. **Practice** by modifying examples or implementing your own variations.
5. **Discuss** – feel free to open an issue or pull request for questions or improvements.
